URL Parameter Details

You can add URL Parameters to create and link to more specific filtered maps. The first parameter must be preceded by a ? and any others must be separated by an &.

eg - http://principalfish.co.uk/electionmaps/?map=prediction&incumbent=conservative&region=wales will only show Conservative seats in Wales on the Nowcast map.

Full Details

Maps - select one

map=prediction : Nowcast

map=predictit : Predict 2022

map=current : Current Parliament

map=election2017 : 2017 Election

map=election2015 : 2015 Election

map=election2010 : 2010 Election

map=election2017_600seat : 600 seat - 2017

map=prediction_600seat : 600 seat- Nowcast

Specifics - select many

incumbent=party : see party codes below

region=region : see region codes below

majlow=integer : Majority/lead low bound

majhigh=integer : Majority/lead high bound

gains=yes or true : Gained seats (only on filters on Election Maps/Prediction)

byelection=yes or true : By-elections (only on filters on Current Parliament)
