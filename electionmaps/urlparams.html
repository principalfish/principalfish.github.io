URL Parameter Details

You can add URL Parameters to create and link to more specific maps. The first parameter must be preceded by a ? and any others must be separated by an &.

eg - http://principalfish.co.uk/electionmaps/?map=prediction&filters=true&incumbent=conservative&region=wales will only show Conservative seats in Wales on the Predicted 2017 map.

eg - http://principalfish.co.uk/electionmaps/?map=election2015&battlegrounds=true&incumbent=labour&challenger=conservative&majlow=5&majhigh=10 will only show Labour held seats where Labour beat the Conservatives by between 5 and 10% in 2015.

IMPORTANT - a map must be selected and ONLY ONE of filters or battlegrounds.

Full Details

Maps - one at a time!

map=prediction : 2017 Prediction

map=predictit : Predict 2017

map=current : Current Parliament

map=election2015 : 2015 Election

map=election2010 : 2010 Election

map=election2015_600seat : 600 seat-2015

map=prediction_600seat : 600 seat-2020

Types - select one or nothing works

filters=yes or true : Filters

battlegrounds=yes or true : Battlegrounds

If filters is active, battlegrounds will not be applied.

Battlegrounds works on 2017 Prediction, Predict 2017, Current Parliament, 2015 Election.

Neither will work on Brexit by Seat.

Specifics

incumbent=party : see party codes below

challenger=party : see party codes below (only for Battlegrounds - must be active)

region=region : see region codes below

majlow=integer : Majority/lead low bound

majhigh=integer : Majority/lead high bound

gains=yes or true : Gained seats (only on filters on Election Maps/Prediction)

byelection=yes or true : By-elections (only on filters on Current Parliament)
